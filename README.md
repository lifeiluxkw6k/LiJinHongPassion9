## 已知问题

在网页上，可以使用中文作为`仓库名称`，但码云提供的 API 并不能用中文。

![仓库和路径](doc/仓库和路径.png)

所以这里使用`路径名称`作为`仓库名称`。已发邮件反馈，等待回复。
